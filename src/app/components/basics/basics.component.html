HTML Basics
  1. What is HTML?

    2. HTML stands for HyperText Markup Language.

    HTML is the standard markup language to create the static web pages.

    HTML contains the series of elements. These elements tells to the browser how to display the content.

  2. What are the basic HTML elements?

    3. <!DOCTYPE html> ----- is instruct to the browser about the version of the html document.

    <html> ----- is the root element of the html document the entire code enclosed inside this element.

    <head> ----- this element contains all the meta information which is useful to load the html document on to the browser.

    <title> ----- this element is used to define the tile to the html document and which appear on the browser tab.

    <body> ----- this element contains all the content which is visible to the user on the web page.

  3. What is html element?

    4. HTML element istruct the browser how to display the content on web page.

    the HTML element contains start tag, content, end tag. Some of the HTML elements doesn't have content and end tag these type of elements are called empty elements or void elements.

  4. What is the tag in HTML?

    5. The tag provides the meaningfull name to the HTML element. Based on the tag names the elemnts are differentiated to display html content onto the web page.

  5. What are the attributes in HTML?

    6. All HTML elements have attribute and these attributes provides the additional information about the HTML element.

    Attributes always specified in the start tag as name value pairs.
